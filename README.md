# 💳 Flipping Card UI Design

## ✨ Descrição do Projeto
Este projeto consiste em uma interface interativa de cartão de crédito com efeito de rotação (flip) ao passar o mouse.  
A proposta é simular a frente e o verso de um cartão.  
Proporciona uma experiência visual moderna e elegante utilizando apenas HTML e CSS.  

O design inclui chip, número do cartão, nome do titular e área de assinatura.  
Criando um layout visual realista e interativo.  

---

## 🎯 Objetivo
- Praticar manipulação de layouts com HTML5  
- Desenvolver estilização avançada com CSS3  
- Aplicar transformações 3D  
- Utilizar efeitos visuais modernos (glassmorphism e blur)  
- Criar interfaces interativas sem uso de JavaScript  

---

## 🧠 Tecnologias Utilizadas
- HTML5  
- CSS3  
- Google Fonts (Poppins)  

---

## 🚀 Funcionalidades
- 🔄 Efeito de rotação 3D ao passar o mouse  
- 💳 Simulação de cartão de crédito (frente e verso)  
- 🎨 Design moderno com efeito glassmorphism  
- 📱 Layout centralizado e responsivo básico  
- 🌈 Elementos visuais com gradiente
