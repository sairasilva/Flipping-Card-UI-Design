💳 Flipping Card UI Design

#

✨ Descrição do Projeto

Este projeto consiste em uma interface interativa de cartão de crédito com efeito de rotação (flip) ao passar o mouse. A proposta é simular a frente e o verso de um cartão, proporcionando uma experiência visual moderna e elegante utilizando apenas HTML e CSS.

O design inclui elementos como chip, número do cartão, nome do titular e uma área de assinatura no verso, criando um layout realista.

#

🎯 Objetivo

O objetivo deste projeto é praticar e demonstrar:

Manipulação de layouts com HTML5
Estilização avançada com CSS3
Uso de transformações 3D
Aplicação de efeitos visuais modernos (glassmorphism e blur)
Criação de interfaces interativas sem uso de JavaScript

#

🧠 Tecnologias Utilizadas

HTML5
CSS3
Google Fonts (Poppins)

#

🚀 Funcionalidades

🔄 Efeito de rotação 3D ao passar o mouse
💳 Simulação de cartão de crédito (frente e verso)
🎨 Design moderno com efeito glassmorphism
📱 Layout centralizado e responsivo básico
🌈 Elementos visuais com gradiente
